Netflix Data Analysis Dashboard 📊

📌 Project Overview

This project focuses on analyzing Netflix content using Power BI. The objective is to understand content distribution, identify trends, and extract meaningful insights from the dataset.


🔧 Tools & Technologies Used

- Power BI (Data Visualization)
- Excel (.xlsx dataset)
- Data Cleaning & Transformation (Power Query)


📊 Dashboard Features

- Total Titles, Ratings, Genres, Locations (KPIs)
- Genre-wise content distribution
- Movies vs TV Shows comparison
- Rating-wise content analysis
- Top 10 countries contributing content
- Year-wise content growth trend


📈 Key Insights

- Movies dominate the platform (~69%) compared to TV Shows
- USA contributes the highest number of titles
- Drama and Documentary genres are the most popular
- Majority of content is rated TV-MA and TV-14 (mature audience)
- Content production increased rapidly after 2015


💡 Recommendations

- Focus more on TV Shows to balance content distribution
- Invest in high-performing genres like Drama and Documentaries
- Expand content production in countries beyond the USA
- Target mature audience content while exploring family-friendly categories
